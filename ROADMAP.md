# ArangoDB GO Driver Roadmap.

- Expected April 2017
- Stable API 
- Supports:
    - ArangoDB 3.1 and up.
    - Single server & cluster.
    - Authentication & secure connections.
- Connection using HTTP(s) + JSON

## Version 1.x 
- Support HTTP(s) + Velocypack connections
- Support Velocystream connections
